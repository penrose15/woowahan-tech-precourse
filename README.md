# woowahan-tech-precourse

### 개요
* 4주간 우아한 테크 코스의 과정을 맛보는 과정

### 배운 점
* 4주간 미션을 통해 깃 컨벤션, 자바 컨벤션 등 가독성 좋은 코드에 대해 깊게 고민해보고 실제 미션에 적용
* 도메인 분리, MVC 패턴 구현 등 객체 지향적인 프로그래밍을 위해 고민해보고 미션에 적용
